(오류가 뜨긴 했지만 파일이 추가는 된듯?)
오류 발생 서로 다른 운영체제 간에 whitespace(공백) 다루는 방법이 차이가 나서 발생하는 오류란다 

git config --global core.autocrlf true 입력으로 해결 
해당 프로젝트에만 적용하고 싶다면 —global 을 빼주면 된다 라고 한다
